Zeppe-Lin v0.99 — Release Notes

Released May 6, 2023. 1st Zeppe-Lin Release.

1 Introduction

Zeppe-Lin v0.99 is the first stable release.
It provides a selected subset of fixes and updates for the beta release from 2023‑01‑02.

2 Upgrade & Compatibility

No compatibility changes are introduced.
A system update should proceed without issues:

# as root
pkgman sysup --deps --depsort

3 What’s New

  • pkgman.conf: added example showing how to renice build packages.

  • Screen blanking timeout made configurable via the BLANKTIME variable in rc.conf. See rc.conf(5).
