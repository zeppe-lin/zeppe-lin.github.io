NAME

pkgman-lock - lock the package(s)

SYNOPSIS

pkgman lock PKGNAME...

DESCRIPTION

Lock the packages and do not update them in a pkgman-sysup(8) operation.

FILES

/var/lib/pkg/locked

Database of locked packages.

EXIT STATUS

0 if successfully locked, 255 otherwise.
