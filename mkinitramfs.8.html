NAME

mkinitramfs - tiny initramfs generator written in POSIX shell

SYNOPSIS

mkinitramfs [OPTION]...

DESCRIPTION

mkinitramfs is a simple, extensible and modular initramfs generator tool written with focus on portability.

OPTIONS

Mandatory arguments to long options are mandatory for short options too.

-o, --output=FILE

Specify output location where initramfs image will be stored.

-c, --config=FILE

Specify config location.

Default is /etc/mkinitramfs/config.

-k, --kernel=VERSION

Specify kernel version. Useful for bootstrapping future system with modular kernel.

Default is current kernel version.

-m, --modules=PATH

Specify directory where modules stored.

Default is /lib/modules.

This option has no effect if monolith or hostonly were enabled in config.

-H, --hooks=PATH

Specify additional hooks location.

Default is /usr/share/mkinitramfs/hooks and /etc/mkinitramfs/hooks.

-D, --helper=FILE

Specify location to device helper.

Default is /usr/share/mkinitramfs/device-helper.

-I, --init=FILE

Specify location to init script.

Default is /usr/share/mkinitramfs/init.

-d, --debug

Enable debug mode. Useful for reporting bugs in build stage.

-f, --force

Forcefully overwrite initramfs image.

NOTES

If you have modular kernel and you use busybox' modprobe to handle modules, you must note that busybox' modprobe doesn't have ability to install soft dependencies of modules (i.e. modules.softdep). You must install them yourself using hooks. See mkinitramfs.hooks(7) how to do it.
