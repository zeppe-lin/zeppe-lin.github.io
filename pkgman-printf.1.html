NAME

pkgman-printf - print formatted list of available packages

SYNOPSIS

pkgman printf [--sort=FORMAT] [--regex] [--filter=PKGNAME] FORMAT

DESCRIPTION

Print formatted list of available packages. It looks in all packages sources directories specified in /etc/pkgman.conf as pkgsrcdir directories.

OPTIONS

--sort=FORMAT

The FORMAT string (see below) that is used to sort the output.

--regex

Interpret filter argument as a regular expression.

--filter=PKGNAME

Filter listing packages by name. The filter option argument may be the string, wildcard or regex.

FORMAT

The string that contains variables, which are replaced like this:

%n

name

%p

path

%v

version

%r

release number

%d

description

%e

dependencies

%u

url

%P

packager

%M

maintainer

%R

has readme (yes/no)

%A

has pre-install script (yes/no)

%B

has post-install script (yes/no)

%C

has pre-remove script (yes/no)

%D

has post-remove script (yes/no)

%l

is locked (yes/no)

%i

is installed (yes/no/diff); "diff" means the package is installed and different package version is available in the packages sources directories.
