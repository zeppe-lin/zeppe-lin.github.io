INTRODUCTION

Zeppe-Lin is a lightweight source-based GNU/Linux distribution for the x86-64 architecture targeted at experienced Linux users. The primary focus of this distribution is keep it simple, which is reflected in a straightforward tar.gz-based package system, BSD-style initscripts, and a relatively small collection of trimmed packages. The secondary focus is utilization of new Linux features and recent tools and libraries. Zeppe-Lin also has a pkgsrc system which makes it easy to install and upgrade applications.

DOWNLOAD

Before downloading Zeppe-Lin please read the handbook(7) to make sure your hardware is supported and to make sure you know how to install it.

Then check rootfs if you want prebuilt rootfs image, mkrootfs if you want to build (custom) rootfs image, and dockerfile if you want to build (custom) docker image.
