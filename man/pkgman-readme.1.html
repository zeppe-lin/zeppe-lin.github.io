NAME

pkgman-readme - print the package's README

SYNOPSIS

pkgman readme PKGNAME

DESCRIPTION

Print the package's README file if it exists.

ENVIRONMENT

If the following environment variable exists it will be utilized by pkgman(1):

PAGER

If the package source directory has a README file, then pkgman will try to open it using a PAGER. If PAGER is unset, then pkgman prints the README to stdout.

EXIT STATUS

0 if success, 255 otherwise.
