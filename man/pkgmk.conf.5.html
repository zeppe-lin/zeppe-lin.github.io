NAME

pkgmk.conf - configuration file for pkgmk

SYNOPSIS

/etc/pkgmk.conf

DESCRIPTION

This manual page describes the format of the pkgmk.conf file. This file is a simple shell file that configures pkgmk(8). The file consists of a number of variable assignments of the form option=value. Comments can be specified by putting a hash (#) symbol as the first character on the line.

If some option is not used (commented out or not included in the configuration file at all) pkgmk(8) will take a default action.

The different configuration options are:

export CFLAGS="STRING"

Set C compiler options.

Default: none.

export CXXFLAGS="STRING"

Set C++ compiler options.

Default: none.

export JOBS="NUMBER"

Set the number of commands to run simultaneously when building a package.

Default: none.

export MAKEFLAGS="STRING"

Set make(1) options.

Default: none.

PKGMK_SOURCE_MIRRORS="STRING ..."

Set mirrors to check and download source archives from. If the URI contains space symbol, replace it by %20.

Default: none.

PKGMK_SOURCE_DIR="STRING"

Set directory for downloaded source archives.

Default: directory of Pkgfile.

PKGMK_PACKAGE_DIR="STRING"

Set directory for built packages.

Default: directory of Pkgfile.

PKGMK_WORK_DIR="STRING"

Set directory for building packages.

Default: 'foo/work', where foo is the directory of the Pkgfile.

PKGMK_DOWNLOAD="STRING"

If set to yes, pkgmk(8) will download the source archives if necessary. Valid strings are yes and no.

Default: no.

PKGMK_DOWNLOAD_PROG="STRING"

Use specified program to download source archives. Valid strings are aria2c, curl and wget.

Default: curl.

PKGMK_ARIA2C_OPTS="STRING"

Additional options for aria2c(1), which is used by pkgmk(8) to download all files.

Default: none.

PKGMK_CURL_OPTS="STRING"

Additional options for curl(1), which is used by pkgmk(8) to download all files.

Default: none.

PKGMK_WGET_OPTS="STRING"

Additional options for wget(1), which is used by pkgmk(8) to download all files.

Default: none.

PKGMK_IGNORE_FOOTPRINT="STRING"

If set to yes, pkgmk(8) will not perform a footprint check of the built package. Valid string are yes and no.

Default: no.

PKGMK_IGNORE_MD5SUM="STRING"

If set to yes, pkgmk(8) will not perform a md5sum check of the source archives. Valid string are yes and no.

Default: no.

PKGMK_IGNORE_NEW="STRING"

If set to yes, pkgmk(8) will ignore new files in a footprint mismatch. Valid string are yes and no.

Default: no.

PKGMK_NO_STRIP="STRING"

If set to no, pkgmk(8) will strip built binaries. Valid string are yes and no.

Default: no.

PKGMK_COMPRESSION_MODE="STRING"

Option to select the mode used to compress the packages. Valid strings are gz, bz2, xz and lz.

Default: gz.

EXAMPLE

#
# /etc/pkgmk.conf: pkgmk(8) configuration
#
# See pkgmk.conf(5) for more information.
#

export CFLAGS="-O2 -march=x86-64 -pipe -fdiagnostics-color=always"
export CXXFLAGS="$CFLAGS"
export JOBS=$(getconf _NPROCESSORS_ONLN)
export MAKEFLAGS="-j $JOBS"

case $PKGMK_ARCH in
"64"|"")
        ;;
"32")
        export CFLAGS="$CFLAGS -m32"
        export CXXFLAGS="$CXXFLAGS -m32"
        export LDFLAGS="$LDFLAGS -m32"
        export PKG_CONFIG_LIBDIR="/usr/lib32/pkgconfig"
        ;;
*)
        echo "Unknown architecture selected! Exiting."
        exit 1
        ;;
esac

# End of file.
