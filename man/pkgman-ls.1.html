NAME

pkgman-ls - print out a listing of the package's directory

SYNOPSIS

pkgman ls [--path] PKGNAME...

DESCRIPTION

Prints out a listing of the package's directory from the packages sources.

OPTIONS

--path

Show path info.

EXIT STATUS

0 if success, 255 if package source is not found.
