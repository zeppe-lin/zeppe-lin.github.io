NAME

pkgdiff - compare software packages' footprints

SYNOPSIS

pkgdiff [OPTION] FILE1 FILE2

DESCRIPTION

pkgdiff searchs which files are new and which are removed between two software packages.

pkgdiff operates on binary packages and requires pkginfo(1).

OPTIONS

-c, --color

Color output.

-v, --version

Print version and exit.

-h, --help

Print help and exit.

EXIT STATUS

0 if success, >0 otherwise.

SEE ALSO

pkginfo(1)
