NAME

pkgman-list-locked - print list of locked packages

SYNOPSIS

pkgman list-locked [-v|-vv] [--path]

DESCRIPTION

Print list of locked packages. "Locked" are packages that will not update while performing a system update.

OPTIONS

-v

Show package version in the listing.

-vv

Show package version and description in the listing.

--path

Show package source path in the listing.

FILES

/var/lib/pkg/locked

Database of locked packages.
