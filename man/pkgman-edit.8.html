NAME

pkgman-edit - edit the package's file

SYNOPSIS

pkgman edit PKGNAME [FILE]

DESCRIPTION

Edit the package FILE from the package source directory using the editor specified in the $EDITOR environment variable. If FILE is not specified, Pkgfile is used.

ENVIRONMENT

If the following environment variable exists it will be utilized by pkgman(1):

EDITOR

If the package's source directory has a FILE, then pkgman will try to open it using a EDITOR. If EDITOR is unset, then pkgman will fail.

EXIT STATUS

0

if success

255

if environment variable EDITOR is unset or package source is not found

>0

EDITOR execution exit status
