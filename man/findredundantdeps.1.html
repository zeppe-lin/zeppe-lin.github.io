NAME

findredundantdeps - find (un)necessary dependencies of a package source

SYNOPSIS

findredundantdeps [OPTION] PKGNAME

DESCRIPTION

findredundantdeps print the full recursive dependencies of a given package displaying the dependencies that are already contained in the depending package.

findredundantdeps operates on packages sources in the pkgsrc tree and requires pkgman(1).

OPTIONS

-u, --unique

Sort and display only unique (required) packages.

-v, --version

Print version and exit.

-h, --help

Print help and exit.

SEE ALSO
