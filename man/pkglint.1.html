NAME

pkglint - detect common mistakes and stylistic issues in pkgsrc package definitions

SYNOPSIS

pkglint [OPTION...] [PKGSRCDIR...]

DESCRIPTION

pkglint runs a series of checks on one or more pkgsrc directories to detect common mistakes and stylistic issues in pkgsrc package definitions. pkglint follows the requirements of Pkgfile(5), but also contains additional checks.

OPTIONS

-d, --dead-links

Check Pkgfile and README files for dead links.

-i, --invalid

Check .footprint file for invalid directories, like /usr/local, /usr/share/locale, /usr/share/doc, /usr/share/info, etc.

-j, --junk

Check .footprint file for junk files, like README, TODO, NEWS, etc.

-s, --suid-sgid

Check .footprint file for SUID/SGID files and directories.

-w, --world-writeable

Check .footprint file for world-writeable files and directories.

-v, --version

Print version and exit.

-h, --help

Print help and exit.

EXIT STATUS

0 if success, >0 otherwise.
