# Zeppe-Lin v0.99.1 - Release Notes {.unnumbered}

Released December 11, 2023.  2nd Zeppe-Lin Release.

![Artwork for v0.99.1](https://github.com/zeppe-lin/artwork/blob/master/v0.99.1.png?raw=true)

# Introduction

Zeppe-Lin v0.99.1 is a maintenance release.
It provides selected fixes and updates for [v0.99](v0.99.html) and
introduces two compatibility changes.

# Upgrade & Compatibility

## pkgutils footprint file changes

Performance optimizations in `pkgutils` altered the generation and
sorting order of `.footprint` files.
This may affect system updates via `pkgman-sysup(8)`.
Perform a forced upgrade of `pkgutils` and `pkgmk` before running a
system upgrade:

```sh
# as root
pkgman update -f --deps --depsort pkgmk pkgutils &&
rejmerge
```

## traceroute conflicts with inetutils

A new `traceroute` package replaces the buggy `traceroute(8)` from
`inetutils`.
Update `inetutils` before installing `traceroute`:

```sh
# as root
pkgman update --deps --depsort inetutils
pkgman install --deps traceroute
```

# What's New

## Notable Fixes

- `pkgsrc-system/ntfs-3g`  
  Added `fsck.ntfs`, a wrapper script around `ntfsfix(8)` that
  silently discards any unsupported flags from `/etc/rc`'s `fsck(8)`.
  Also added `fsck.ntfs(8)` manual page.

## Package Changes

<!-- generated by ./gen_release_pkgchanges.sh v0.99.1 -->

### pkgsrc-core

- New: `seedrng` `traceroute` 
- Removed: `capstone` 
- Notes:
  - `capstone` moved to `pkgsrc-system`
  - `/etc/pkgadd.conf` updated to silently upgrade `/etc/revdep.d`
    files
  - `rejmerge(8)` allows redefining `PAGER` and `EDITOR` from
    command-line
  - fixes and cleanups for `sysklogd`'s `rotatelog`
  - added `rotatelog(8)` manual page to `sysklogd` package source

### pkgsrc-system

- New: `alsa-ucm-conf` `capstone` `rdfind` 
- Removed: *None*
- Notes:
  - `tree` switched to the maintained fork
  - `capstone` imported from the `pkgsrc-core`
  - `ntfs-3g`: added `fsck.ntfs` (wrapper for `ntfsfix(8)`) and
    `fsck.ntfs(8)` manual page
  - `lm-sensors`: configs moved from `/usr/etc` to `/etc`
  - `freetype`: no longer pollutes `/etc/profile.d`
  - `p11-kit`: added `update-ca-trust(1)` manual page
  - `rdfind` required by `linux-firmware`

### pkgsrc-xorg

- New: `gxmessage` `ttf-font-jetbrains-mono`
  `ttf-font-jetbrains-mono-nerd` `vulkan-headers` `vulkan-loader`
  `vulkan-tools` `wmctrl` `xorg-libxpresent` `xorg-xcompmgr` 
- Removed: *None*
- Notes:
  - `xorg-libxpresent` required by `mpv`.
  - Vulkan set required by `mesa` for `zink` support.

### pkgsrc-desktop

- New: `alsa-firmware` `alsa-plugins` `broodwar-cursor-theme`
  `firefox-darkreader-bin` `firefox-vimium-bin` `json-glib` `ladspa`
  `lsw` `lxrandr` `pick` `ratpoison` 
- Removed: *None*
- Notes:
  - `json-glib` required by `zathura`
