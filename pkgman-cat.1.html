NAME

pkgman-cat - print out the package's file

SYNOPSIS

pkgman cat PKGNAME [FILE]

DESCRIPTION

Prints out the package file from the package source directory. If FILE is not specified, Pkgfile is used.

ENVIRONMENT

If the following environment variable exists it will be utilized by pkgman:

PAGER

If the package's source directory has a FILE, then pkgman(1) will try to open it using a PAGER. If PAGER is unset, then pkgman(1) prints the FILE to stdout.

EXIT STATUS

0 if success, 255 if package's source directory (or package's file) is not found.
