NAME

mkrootfs - make root filesystem for chroot installation

SYNOPSIS

mkrootfs [COMMAND] [OPTIONS]...

DESCRIPTION

mkrootfs builds customized root filesystem for chroot installation.

COMMANDS

rootfs

Prepare rootfs directory.

revdep

Check rootfs directory for missing libraries.

tar

Compress rootfs directory (prepare tarball).

sign

Sign prepared tarball.

OPTIONS

Mandatory arguments to long options are mandatory for short options too.

-r, --rootfs=PATH

Set path to rootfs directory.

-t, --tar=FILE

Set path to compressed rootfs tarball.

-l, --log=FILE

Set path to log file for pkgman(1).

-p, --packages="PKG1 PKG2..."

Set default packages to extract into the rootfs directory.

-c, --config=FILE

Use alternative configuration file.

Default: /etc/mkrootfs/config.

-x, --pkgmk-config=FILE

Use alternative pkgmk(8)'s configuration file.

Default: /etc/pkgmk.conf.

-y, --pkgman-config=FILE

Use alternative pkgman(1)'s configuration file.

Default: /etc/pkgman.conf.

FILES

/etc/mkrootfs/config

Global mkrootfs' configuration. See mkrootfs.config(5) for more information.

/etc/pkgmk.conf

Configuration file for invocated pkgmk.

/etc/pkgman.conf

Configuration file for invocated pkgman.

EXIT STATUS

"0"

No error occurred.

"1"

A general error has occurred.

"2"

An error occurred during building rootfs directory.

"3"

An error occurred during checking rootfs directory for missing libraries.

"4"

An error occurred during compressing rootfs directory (preparing the tarball).

"5"

An error occurred during signing prepared tarball.

SEE ALSO
