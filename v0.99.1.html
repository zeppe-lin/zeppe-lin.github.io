Zeppe-Lin v0.99.1 - Release Notes

Released Dec 11, 2023. 2nd Zeppe-Lin Release.

1 Introduction

The Zeppe-Lin project is happy to announce the second stable release v0.99.1! It represents a selected subset of fixes and updates for v0.99 and has one small (and not painful) breaking compatibility change.

2 Incompatible Changes

  • pkgutils Footprint File Changes
    During performance optimizations for pkgutils, the generation and sorting order of footprint files has been changed. This may break system updates via pkgman-sysup(8). To avoid this, perform a forced upgrade of pkgutils and pkgmk before running a system upgrade:

    # as root
pkgman update -f pkgmk pkgutils &&
rejmerge

  • New traceroute conflicts with inetutils
    New package traceroute, introduced to replace buggy traceroute(8) utility provided by inetutils. Make sure you update inetutils before you install traceroute.

3 New Features

Mostly fix:

  • pkgsrc-system/ntfs-3g
    Added fsck.ntfs, a wrapper script around ntfsfix(8) that silently discards any unsupported flags from /etc/rc’s fsck(8). Also added fsck.ntfs(8) manual page.

3.1 Package Changes

3.1.1 pkgsrc-core

  • New: seedrng traceroute
  • Removed: capstone
  • Notes:
    • capstone moved to pkgsrc-system
    • /etc/pkgadd.conf updated to silently upgrade /etc/revdep.d files
    • rejmerge(8) now allows redefining PAGER and EDITOR from command-line
    • small fixes and cleanups for sysklogd’s rotatelog
    • added rotatelog(8) man page to sysklogd

3.1.2 pkgsrc-system

  • New: alsa-ucm-conf capstone rdfind
  • Removed: None
  • Notes:
    • tree switched to the maintained fork
    • capstone imported from the pkgsrc-core
    • ntfs-3g: added fsck.ntfs (wrapper for ntfsfix(8)) and fsck.ntfs(8) man page
    • lm-sensors: configurations moved from /usr/etc to /etc
    • freetype: no longer pollutes /etc/profile.d
    • p11-kit: added update-ca-trust(1) man page
    • rdfind required by linux-firmware

3.1.3 pkgsrc-xorg

  • New: gxmessage ttf-font-jetbrains-mono ttf-font-jetbrains-mono-nerd vulkan-headers vulkan-loader vulkan-tools wmctrl xorg-libxpresent xorg-xcompmgr
  • Removed: None
  • Notes:
    • xorg-libxpresent required by mpv.
    • Vulkan set required by mesa for zink support.

3.1.4 pkgsrc-desktop

  • New: alsa-firmware alsa-plugins broodwar-cursor-theme firefox-darkreader-bin firefox-vimium-bin json-glib ladspa lsw lxrandr pick ratpoison
  • Removed: None
  • Notes:
    • json-glib required by zathura
