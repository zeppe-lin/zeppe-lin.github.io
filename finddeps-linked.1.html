NAME

finddeps-linked - find linked dependencies for a package

SYNOPSIS

finddeps-linked [OPTION] PKGNAME

DESCRIPTION

finddeps-linked lists the dependencies of the installed package's dynamic executables and libraries.

finddeps-linked operates on installed packages and requires ldd(1), pkginfo(1) and pkgman(1).

OPTIONS

-v, --version

Print version and exit.

-h, --help

Print help and exit.
